Command

setupchat

Creates a channel #talk-with-vibhi where you can chat with Vibhi
v!setupchat

roleplay [rp]

Lists all roleplay commands which you can use
v!roleplay
v!{roleplay command}

avatar [av, pfp]

Shows the avatar of the users that are mentioned can also show multiple avatars together
v!avatar @weeblet~kun
v!pfp @weeblet~kun @minnxx

splitimg

Splits an image or a gif into two equal parts (1:1 ratio) useful for avatar sharing must attach an image file with this command
v!splitimg {attach an image}

prefix

Add and remove custom prefixes
options : add, remove v!prefix add >>
v!prefix remove >>

anilist

Shows anilist stats for a user
v!anilist {anilist username}

mal

Shows My Anime List stats for a user
v!mal {mal username}
